Q1: 欄位說明
A1: 
您好，
資料可分成 3 大類
籌碼：外資買/賣張、買/賣超第N名前T天買/賣張(N=1~15, T=1~20)、庫存，等一般做股市分析時會被分類在籌碼面的欄位
價量：K(9)、D(9)、月K(9)、個股T天報酬率，等一般做股市分析時會被分類在價量面的欄位
基本：單筆合併營收(千)、淨值、營收成長率，等一般做股市分析時會被分類在基本面的欄位
Training Data 一共有 10,214 個欄位，其中一欄為 ID，一欄為 y 標記 飆股，其餘的 10,212 個欄位則為 Feature 欄位。
單欄位的解釋可直接參考欄位名稱，欄位名稱在設計時都採常見用語，應該是蠻好理解的，若對欄位定義上有什麼問題也歡迎提出詢問，謝謝。


Q2: 所以一筆資料中，y代表這支股票過去一個月的表現是否符合飆股的定義，相對的那些features也都是過去一個月的值這樣嗎?
以及那dataset中會有重複的股票，但是是不同時間段的資料嗎?
A2: y 代表的是未來一定時間會是飆股, 而不是過去一個月.
dataset 裡面重複的股票就是指不同時段的資料沒錯.

Q3: 想問競賽規則第五點提到得獎者需提供「額外資料集」以及取得方式，是否代表可以使用非訓練資料來訓練模型，是否也代表可以使用額外資料集訓練的預訓練模型呢
A3: 是, 比賽並不想禁止自己蒐集額外的資料這種 ML 中常見且有效的手段, 只是要提供取得方法來確認公平性. 假設有某個特徵是透過人工憑經驗標註的話, 就不是比賽希望看到的

Q5: 
請問主辦方關於下述欄位定義
個股券商分點區域分析_買超1公司區域密度
個股券商分點區域分析_買超2分點區域密度
個股券商分點區域分析_買超5與公司接近比例
個股券商分點區域分析_近1日買超5分點間接近比例
A5:
個股券商分點區域分析_買超1公司區域密度：買超前一名券商分點與公司間的區域密度分數
個股券商分點區域分析_買超2分點區域密度：買超前二名券商分點間的區域密度分數
個股券商分點區域分析_買超5與公司接近比例：買超前五名券商分點中，與公司距離三公里以內的家數比例
個股券商分點區域分析_近1日買超5分點間接近比例：買超前五名券商分點中，彼此間距離三公里以內的比例
